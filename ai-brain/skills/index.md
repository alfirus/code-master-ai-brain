# Skills Directory

This directory contains capabilities and how-to guides for specific tasks.

## How to Use

1. Identify the task you need to perform
2. Find the relevant skill
3. Follow the skill's steps and guidelines
4. Adapt as needed for your context

## Skill Format

Each skill includes:

```markdown
# [Skill Name]

## Purpose
What this skill helps accomplish

## Prerequisites
What you need before using this skill

## Steps
1. Step one
2. Step two
3. ...

## Examples
Practical demonstrations

## Tips
Best practices and common pitfalls

## Related Skills
Other skills that complement this one
```

## Using Skills

### Loading a Skill
1. Read the skill file
2. Understand its purpose and prerequisites
3. Keep it accessible during the task

### Applying a Skill
1. Follow steps in order
2. Adapt to your specific context
3. Reference examples as needed

### Combining Skills
- Skills can be used together
- Check related skills for comprehensive approaches
- Build skill chains for complex tasks

## Adding Skills

To contribute a new skill, see `contributions/CONTRIBUTE.md`.

Good skill candidates:
- Repeatable processes
- Complex tasks with clear steps
- Techniques that improve quality
- Methods that save time
