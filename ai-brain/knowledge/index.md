## Knowledge Format

Each knowledge file should include:
- **Topic**: What the knowledge covers
- **Summary**: Brief overview
- **Details**: Full information
- **Examples**: Practical applications
- **Sources**: Where this knowledge came from
- **Last Updated**: When it was last verified
